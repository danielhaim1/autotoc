AutoToc

AutoToc is an NPM package for generating and managing table of contents with scroll highlighting for web content.

Learn more at https://github.com/danielhaim1/autotoc

Dependencies:

@danielhaim/modulator
@danielhaim/slugify

Installation

$ npm i @danielhaim/autotoc

Why AutoTOC?

AutoTOC automates the process of generating a Table of Contents, offering scroll highlighting and easy navigation through lengthy documents.

Getting Started

Quick setup instructions to get AutoTOC running on your project.

$ npm install @danielhaim/autotoc

Features

Automatic TOC Generation

Automatically generate a table of contents based on the document's headings.

Scroll Highlighting

Highlights sections in the TOC as you scroll through the document.

Customization Options

Customize the appearance and behavior of the TOC via a simple set of options.

Advanced Usage

Dive deeper with advanced configurations and methods to tailor the TOC to fit your needs.
