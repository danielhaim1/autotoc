RandomUtil API

RandomUtil is an NPM package for easily populating DOM elements with random data, perfect for testing and prototyping web interfaces. Supports various data types including text, images, and dates

Get Started

npm install @danielhaim/randomutil

Usage

const UNSPLASH_API_KEY = '[your-unsplash-api]';

RandomContentManager

  • Parameter: The category ID to select the corresponding data arrays.
const randomContent = new RandomContentManager(1);

RandomController

const randomController = new RandomUtilController({});

RandomTag, RandomTitle, RandomExcerpt

randomController.randomTag(randomContent.randomTags);
    randomController.randomTitle(randomContent.randomTitles);
    randomController.randomExcerpt(randomContent.randomExcerpts);

RandomReadTime

randomController.randomReadTime();

RandomDate

randomController.randomDate("Y/m/d");

RandomImages

const randomImageCount = document.querySelectorAll("[data-random='img']").length;
    randomController.randomImages({ count: randomImageCount, query: "nature", orientation: "portrait", accessKey: UNSPLASH_API_KEY });

RandomAvatar

randomController.randomAvatar({ avatarOptions: { variant: "defaultVariant" } });

randomImages Method

  • This method triggers the imageManager to generate and update elements with a random image.
  • It targets [data-random="img"] attribute managed by the imageManager and updates their content with a random image.
  • By default, the randomImages() method will generate a random image from the imageManager image list.
  • However, you can pass in a custom query to the randomImages() method to generate a random image based on a custom query.
  • See the image query section below for more information on how to format image queries.

Parameters:

  • count (optional, default = 12): The number of images to fetch.
  • query (optional, default = "nature"): The search term used to query images from Unsplash.
  • orientation (optional, default = "landscape"): The orientation of the images ("landscape", "portrait", or "squarish").
  • accessKey (required): The Unsplash API access key.

randomAvatar Method

  • This method triggers the avatarManager to generate and update elements with a random avatar.
  • It targets [data-random="avatar"] attribute managed by the avatarManager and updates their content with a random avatar.
  • By default, the randomAvatar() method will generate a random avatar from the avatarManager avatar list.
  • However, you can pass in a custom avatar variant to the randomAvatar() method to generate a random avatar based on a custom variant.
  • See the avatar variant section below for more information on how to format avatar variants. 
    const rc = new RandomController();
    rc.randomAvatar({
      avatarOptions: {
        variant: "smile",
        colors: ["#FF5733", "#33FF57", "#5733FF"],    
        square: true,
        size: 128,
      },
    });

randomDate Method

  • This method triggers the dateManager to generate and update elements with a random date.
  • It targets [data-random="date"] attribute managed by the dateManager and updates their content with a formatted date.
  • By default, the randomDate() method will generate a random date in the format of M j, Y (Eg., Dec 21, 2019).
  • However, you can pass in a custom format to the randomDate() method to generate a random date in a custom format.
  • See the date formatting section below for more information on how to format dates.
